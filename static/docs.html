How AI Rank works and how to use it.

Overview

AI Rank tests how often your website is mentioned when ChatGPT answers category-related prompts. You enter your website URL and product category; we run many prompts and return a visibility score plus suggestions.

Using the web app

  1. What's your website? — Enter your domain (e.g. example.com).
  2. What does your website do? — Describe your product (e.g. pdf editor, productivity app).
  3. Click Check My AI Ranking and wait while we run 20–50 prompts against ChatGPT.
  4. Review your visibility score, which prompts mentioned you, and suggestions to improve.

Understanding the results

  • Visibility score — Percentage of prompts where your site was mentioned.
  • Keyword results — Each prompt we tested, with ✓ if you were mentioned.
  • Suggestions — Keyword swaps and tips to improve AI visibility.

API

If you prefer to use the API directly:

POST /analyse

{
  "website": "example.com",
  "category": "pdf editor"
}

Response includes visibility_score, prompts_tested, mentions_found, prompt_results, and suggestions.

Interactive API reference: /docs (OpenAPI/Swagger).

Which AI models?

Currently we test against ChatGPT only. Support for Claude, Gemini, and others is planned.
